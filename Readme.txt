Cara Penggunaan:
1. Install Composer.
2. Setelah di install buka "Power Shell" menggunakan Run.
3. Cek apakah composer berjalan.
4. Setelah composer berjalan, arahkan dalam power shell menuju folder dimana source code disimpan, (ex: C://wampserver/www/nama_folder/...)
4. Jalankan Composernya ketik : "php spark serve"
5. Pastikan phpmysql berjalan pada server yang anda Install sebelumnya baik XAMPP maupun Wamp.
6. Jalankan melalui browser.
